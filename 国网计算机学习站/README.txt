国网计算机类学习站（离线版）

使用方式：
1. 双击 index.html 即可打开，不需要联网，也不需要安装软件。
2. 首页分为“知识点、模拟卷、专项练习”三个子页面。
3. 知识点按“科目 → 章节 → 小节 → 知识点”分级，可搜索和标记掌握。
4. 模拟卷共 3 套、400 道结构化题；两套含答案的试卷提交后自动判分，答错会标出正确选项；模拟题1只保存选择，暂不判分。
5. 专项练习共 5 类、708 道结构化题。扫描件中的参考答案原文保留在折叠栏中。
6. 知识点、试卷和专项练习均可跳回原 PDF 页，并保留清理广告后的分页原文用于核对公式、图表或 OCR 内容。
7. 学习进度、题目选择和深色模式保存在当前浏览器中。

页面数据：
- 知识点：6 科、46 章、142 小节
- 章节练习：875 道
- 模拟卷：3 套、400 道
- 专项练习：5 类、708 道

说明：
网页不依赖外部字体、框架或网络资源。扫描 PDF 已经过中文 OCR、广告页眉过滤和题号层级复核；遇到公式或图形题时，请使用“打开原 PDF”核对原始页面。
